# Total Destruction
A Python pygame created by Christopher Ross with love for his children Sawyer and Elle!

Author: Christopher Ross

Published: 2022

# How to play
1) Install `Python3`
2) Setup with `pip install -r requirements.txt`
3) run command `python td.py`
